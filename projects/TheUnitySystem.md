The Unity System was created a classmate and myself. When tasked to create a progject that implements a user database we chose to create a system that allowed Students, Teachers, and Parents to fully interact with their school from the recruitment process all the way through attending. My partner and I both have history working in educvation and using various sytems. our goal with our project was to come up with one universal system that acts as a one stop shop for all individuals in education.
